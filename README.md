# Email-Bot

### Installation

1. clone application:
```git clone https://github.com/BhavinRaichura/Email-Bot.git```

2. Install python

3. create vertual environment
```python -m venv venv```

4. start venv
```. venv/bin/activate```

5. Install required libreries from requirements.txt
```pip install -r requirements.txt```

6. Run the application
```python app.py```
